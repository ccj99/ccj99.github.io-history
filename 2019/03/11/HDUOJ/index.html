题目：输入 k ， 求一个最小的非负整数 a ，使得对任意 x 有 f(x)=5*x^13+13*x^5+k*a*x % 65 = 0

费马小定理： 假如p是质数，且gcd(a,p)=1，那么 a(p-1)≡1（mod p），例如：假如a是整数，p是质数，则a,p显然互质(即两者只有一个公约数1)，那么我们可以得到费马小定理的一个特例，即当p为质数时候， a^(p-1)≡1(mod p)。

将 f(x) 转化为 f(x)=(5*x^12+13*x^4+k*a)*x ，要对任意 x 都有 f(x) % 65 == 0 ，则有 (5*x^12+13*x^4+k*a) % 65 = 0 ，将这个问题分解为 (5*x^12+13*x^4+k*a) % 5 == 0 && (5*x^12+13*x^4+k*a) % 13 == 0， 根据费马小定理，对第一个式子有 5*x^12==5*x^(13-1)%13==5， 所以第一个式子成立，等价于 k*a % 13 == 8 ，同理第二个式子成立等价于 k*a % 5 == 2。这样我们就得到了判别条件，可以快速筛选出 a 。
