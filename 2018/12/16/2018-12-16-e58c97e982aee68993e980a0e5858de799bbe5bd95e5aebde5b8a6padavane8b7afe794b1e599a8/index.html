北邮打造免登录宽带Padavan路由器

2018-12-16 网络配置 0 Comments

硬件接法

外网接WAN口，内网接LAN口，与平常路由器相同，与官方指导不同

基本设置

开启DHCP
外网链接类型基本上是随便设置，我选的是IPoE：动态IP，选啥应该都能用
这时候打开认证网关，登陆就可以了，再连到路由器的设备都可以免登录上网

IPv6问题

这时候还不能访问BT，路由器也无法获取IPv6的地址，打开IPv6设置，改成如图：

DNS是网上找的，如下：

2001:da8:202:10::36
2001:da8:202:10::37

这时候，路由器应该可以获取到IPv6的地址了，但是还是上不去bt，因为路由器还不能给下级设备分发V6地址
ssh连接你的路由器管理员
安装opkg（先试试安装了没有）

wget http://pkg.entware.net/binaries/mipsel/installer/installer.sh && chmod +x installer.sh  && ./installer.sh

接着执行

opkg update
opkg install 6relayd
6relayd -d -A eth2.2 br0

eth2.2代表路由器的WAN口，br0代表整个内网，一般都不用改，可以通过ifconfig命令验证，至此，可以访问BT了，所有配置结束

重启后以上设置会失效，开机自动开启脚本为：

#开机自动安装运行 6relayd

export PATH='/etc/storage/bin:/tmp/script:/etc/storage/script:/opt/usr/sbin:/opt/usr/bin:/opt/sbin:/opt/bin:/usr/local/sbin:/usr/sbin:/usr/bin:/sbin:/bin'
export LD_LIBRARY_PATH=/lib:/opt/lib
while ! [ -x "`which opkg`" ]
do
    logger -t "【6relayd】" "Waitting opt install"     
    sleep 3
done
while ! [ -x "`which 6relayd`" ]
do
    logger -t "【6relayd】" "6relayd not found,begin to install it"
    opkg update
    opkg install 6relayd
done
logger -t "【6relayd】" "6relayd has been installed"
6relayd -d -A eth2.2 br0
logger -t "【6relayd】" "6relayd start"

长时间没有IPv6连接6relayd会掉线，把每半个小时访问一次Google加入定时任务即可

*/30 * * * * ping -6 -c 2 ipv6.google.com
